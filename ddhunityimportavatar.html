Importing the avatar

NOTE

Laura_V5 will be used as the default character for this section

1. Using the menu bar, navigate to DDH>Import Folder (to import the entire character folder) or DDH>Import Model (to import a single 3D model)

- Character folders can be downloaded here

Import Folder vs Import Model

After the character folder is imported, everything that is related to the character is contained within the folder, including the base character model (character rig), animation files, atlas videos, texture, etc. When a single 3D model is imported, ensure that the base character mode (character rig) is imported before the character's animation files.

Animations are created separately from the character model as they are used to drive the character rig in Unity. When the character model is modified, the model that all the animations drive is changed, and therefore every single animation file does not need to be changed to adapt to the modifications in the base character model.

2. If the "A material is using the texture asz a normal map" message appears, click Fix Now

3. Imported folder/avatar will be placed in the project menu under Assets>Resources>DDH_Avatars

4. In the Project Menu, navigate to Assets>Resources>DDH Avatars>LauraV5>Animations and click on any .fbx animation file. In the Inspector menu on the right side, select the Animation tab and change the name shown next to a turquoise triangle accordingly and press the enter key. Then click Apply at the bottom of the Inspector. Repeat this step for all animation files.
