DDH Tool - FBX Combiner

FBX Combiner enables combining of the base avatar .fbx file with animation .fbx file(s) and produce one .fbx file

1. Right click within the folder in which you wish to create the FBXCombiner asset

2. Double click on the created asset and in the Inspector enter all the required fields

i. Base Model: drag/find the base .fbx file of the avatar

ii. Output Folder: where to store the created .fbx file with all the animations

a.The path can either be manually entered in or chosen with the Select Output Folder button

b. After choosing the path with the Select Output Folder button, click anywhere within the Inspector to see the path appear in the Output Folder section

iii. Output Model Name: name the .fbx file (ensure that the name is different than the base .fbx file name)

iv. Number of animations to import: how many animation .fbx files to import with the base avatar

v. Animation FBX & Clip Name: drag/find the animation .fbx to combine

vi. Debug message filter: choose the level of message filter to show on the console

a. None

b. Low: Logs

c. Medium: Logs and Warnings

d. High: Logs, Warnings, and Errors

3. After entering all the fields, click Combine FBX and the .fbx file should appear within the output folder path you have selected
